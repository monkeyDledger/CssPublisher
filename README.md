# Android 直播、录屏客户端
***

>  支持Android5.0以上
>  sysnew账户登录，暂不支持注册
>  提供设备录屏、摄像直播和文件管理三个主要功能

## 设备录屏
> 因为录屏可能带来安全问题，所以录屏开始前安卓会弹出警告框

* 本地录屏，需设置文件名，且不能与本地已有文件重名
* 录屏直播，基于rtmp协议，推流地址跟sysnew用户名绑定；可以选择直播的同时保存为本地文件
## 摄像直播
> 提供切换分辨率和保存本地录像选择，直播前需确保开启摄像头预览，支持前后摄像头切换


## 文件管理
> 获取所有保存过的本地文件，提供刷新、批量删除和批量上传，点击播放功能
